================================================================================
AUTHENTICATION FLOW
================================================================================

1. User visits /dashboard → middleware checks auth
2. No session → redirect to /login
3. User enters credentials → NextAuth signIn()
4. Valid credentials → session created (JWT)
5. Redirect to /dashboard → middleware allows access
6. Dashboard layout renders with sidebar
7. User data displayed in sidebar (name, email, role)
8. Navigation shows role-based links
9. Logout → signOut() → redirect to /
