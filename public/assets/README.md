# Assets Directory

This directory should contain the following brand assets from the Crema Arena design system:

- `wordmark.svg` - Full Crema Arena wordmark logo
- `monogram.svg` - Crema Arena monogram
- `stamp-seal.svg` - Award badge stamp
- `rings.svg` - Live dot and avatar frame graphics
- `trophy.svg` - Winner trophy icon

These assets are used throughout the application UI.
